## Description

`Button` 컴포넌트는 다양한 스타일 변형을 제공하여 다양한 사용자 인터페이스 요구 사항에 맞게 사용할 수 있습니다.

- `default`: 기본 버튼 스타일을 제공합니다.
- `gray`: 회색 배경의 버튼 스타일을 제공합니다.
- `stroke`: 테두리 스타일이 적용된 버튼을 제공합니다.

각 버튼은 `disabled` 상태일 때 스타일이 변경됩니다, 사용자가 상호작용할 수 없는 상태를 명확하게 표시합니다.
