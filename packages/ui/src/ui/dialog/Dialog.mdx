## Description

`Dialog` 는 사용자에게 메시지를 표시하는 컴포넌트입니다.

- 기본적으로는 합성 컴포넌트로 사용할 수 있습니다.

```tsx
<Dialog>
	<DialogTrigger className='w-full'>
		<Button>Open</Button>
	</DialogTrigger>
	<DialogContent withCloseButton={true}>
		<DialogHeader>
			<DialogTitle>리뷰쓰기</DialogTitle>
			<DialogDescription>
				000고객님! 구매하신 내역이 없어 후기를 작성하실 수 없어요!
			</DialogDescription>
		</DialogHeader>
		<DialogFooter>
			<DialogClose asChild>
				<Button>확인</Button>
			</DialogClose>
		</DialogFooter>
	</DialogContent>
</Dialog>
```

- `Dialog`는 또한 `useDialog`훅을 사용하여 사용할 수 있습니다.
- `footer`는 배열로 버튼을 추가할 수 있습니다.(최대 2개)

```tsx
const CustomHook = () => {
	const { openDialog } = useDialog();

	const onClick = () => {
		openDialog({
			title: "아직 응모기간이\n끝나지 않았어요!",
			description: "지금 당첨자를 발표하면 재추첨이 불가능해요",
			footer: ["추첨하기"],
			justify: "left"
		});
	};

	return <Button onClick={onClick}>Open</Button>;
};
```
