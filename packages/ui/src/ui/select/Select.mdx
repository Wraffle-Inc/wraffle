## Description

`Select` 항목을 선택할 수 있는 드롭다운 메뉴를 제공합니다.

- 단순 Select는 props의 items 타입을 `SelectItem[]`로 전달해야 합니다.

```tsx
interface SelectItem {
  value: string;
  name: string;
}
```

- 그룹 Select는 props의 items 타입을 `GroupSelectItem[]`로 전달해야 합니다.

```tsx
interface GroupSelectItem {
  groupName: string;
  groupItems: SelectItem[];
}
```
